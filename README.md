## Project Overview

A program to predict a home's sell price using data scraped from the internet.

`homeBuyerFire` was written in Python 3.7.

## Getting Started

### Dependencies

- `beautifulsoup4`
- `geopy`
- `requests`
- `pandas`
- `configparser`
